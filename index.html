LAB 9 — Inter-VLAN Communication (Router-on-a-Stick) Lab 9
What is Inter-VLAN?

VLANs are isolated by default. To let different VLANs talk, you need a Layer 3 router with sub-interfaces — one per VLAN. The trunk link carries all VLAN traffic using dot1Q tagging.

This lab uses dot1Q encapsulation on router sub-interfaces. The router port must be UP (no shutdown) and each sub-interface needs encapsulation dot1Q [VLAN-ID].

LAB 10 — DHCP on Router + DNS Server Lab 10
What is DHCP?

DHCP automatically assigns IP addresses to PCs. The router gives out IPs from a defined pool. DNS server IP is also sent to PCs so they can resolve domain names.

Always exclude static IPs (router + DNS server) from the DHCP pool so they don't get reassigned.

Network Plan — 192.168.1.0/24

LAB 11 — RIP Protocol Lab 11
What is RIP?

RIP is a dynamic routing protocol. Routers share routing tables every 30 seconds and pick the path with the fewest hops. Max 15 hops. Always use RIPv2.

Key Facts

Metric: Hop Count
Update: every 30s
Max hops: 15
Port: UDP 520

show ip route symbols

C = Connected
R = RIP learned
[120/1] = distance/hops

LAB 12 — OSPF Protocol (3 Routers) Lab 12
What is OSPF?

OSPF is a link-state routing protocol using cost as metric. Faster than RIP. Uses Dijkstra's algorithm. All routers go in Area 0. Uses wildcard masks (inverse of subnet mask).

Wildcard Masks

Subnet MaskWildcard
255.255.255.00.0.0.255
255.255.255.2520.0.0.3
255.0.0.00.255.255.255
